Geodemographic Segmentation¶

Company Issue:¶

We are having a problem with customers leaving our bank. We need to find out why. Help us understand what we need to do.

Geodemographic segmentation is a multivariate statistical classification technique for discovering whether the individuals of a population fall into different groups by making quantitative comparisons of multiple characteristics with the assumption that the differences within any group should be less than the differences between groups.

def AB_PLOT(exited1, exited2, stayed1, stayed2, category1, category2, on1, on2):
    
    data1 = [exited1, exited2]
    data2 = [stayed1, stayed2]

    f, (ax1, ax2) = plt.subplots(1, 2, figsize=(10,5))

    ## Absolute count
    ax1.bar(range(len(data1)), data1, label = on1, color = '#FF6600')
    ax1.bar(range(len(data2)), data2, bottom = data1, label = on2, color = 'b')
    plt.sca(ax1)
    plt.xticks([0.4, 1.4],  [category1, category2])
    ax1.set_ylabel("Count")
    plt.legend()

    ## Percent
    totals = [i + j for i,j in zip(data1, data2)]
    data1_rel = [i / j * 100 for  i,j in zip(data1, totals)]
    data2_rel = [i / j * 100 for  i,j in zip(data2, totals)]

    ax2.bar(range(len(data1_rel)), data1_rel, label = on1, color = '#FF6600')
    ax2.bar(range(len(data2_rel)), data2_rel, label = on2, bottom = data1_rel, color = 'b')
    plt.sca(ax2)
    plt.xticks([0.4, 1.4],  [category1, category2])
    plt.axhline(20, c = 'w')
    ax2.set_ylabel("Percentage %")
    plt.legend()

    plt.show()
    
def AB_Test(column, on1, on2):
    exited1 = dat[(dat[column] == 1) & (dat['Exited'] == 1)].shape[0]
    stayed1 = dat[(dat[column] == 1) & (dat['Exited'] == 0)].shape[0]

    exited2 = dat[(dat[column] == 0) & (dat['Exited'] == 1)].shape[0]
    stayed2 = dat[(dat[column] == 0) & (dat['Exited'] == 0)].shape[0]
    
    return AB_PLOT(exited1, exited2, stayed1, stayed2, on1, on2, 'Exited', 'Stayed')

# frequentist approach -- Numerical Output
def AB_Chisquare_Test(column):
    dat1 = dat[dat[column] == 0]
    dat2 = dat[dat[column] == 1]
    
    obs = [[len(dat1[dat1.Exited == 0]), len(dat1[dat1.Exited == 1])], 
       [len(dat2[dat2.Exited == 0]), len(dat2[dat2.Exited == 1])]]
    t_stat, p, dof, exp = stats.chi2_contingency(obs)
    return t_stat, p

import pymc as pm

def bayes_AB(column):
    # get count and len for both priors
    data_A = dat[(dat[column] == 0) & (dat.Exited == 1)].shape[0] 
    data_A_len = dat[dat[column] == 0].shape[0]
    data_B = dat[(dat[column] == 1) & (dat.Exited == 1)].shape[0]
    data_B_len = dat[dat[column] == 1].shape[0]
    
    # set up numpy array of ones and zeros 
    data_A = np.r_[[0] * (data_A_len - data_A), [1] * data_A]
    data_B = np.r_[[0] * (data_B_len - data_B), [1] * data_B]
    
    # create priors, have no assumption so use uniform priors
    p_A = pm.Uniform('p_A', lower = 0, upper = .99)
    p_B = pm.Uniform('p_B', lower = 0, upper = .99)
    
    # set up determistic variable to calc delta(difference in samples)
    @pm.deterministic
    def delta(p_A=p_A, p_B=p_B):
        return p_B - p_A
    
    # create likelihoods with Bernoulli dist
    obs_A = pm.Bernoulli('obs_A', p_A, value = data_A, observed = True)
    obs_B = pm.Bernoulli('obs_B', p_B, value = data_B, observed = True)
    
    return pm.MCMC([p_A, p_B, delta, obs_A, obs_B])

The Cumulative Accuracy Profiles (CAP) curve is an approach that allows a business to target their audience. This is built in excel as their is an easy template to create this image and would be a pain to do it with code. This image is read as: the red line indicates random guessing and the yellow is what the classifer predicts. If the bank went after the customers that the logistic regression predicted as the highest probabilities the bank would be able to target much better than the base line. You can compare the two lines and see how much influence the classifer has on the data.
